# ยิงเบอร์ฟรี โดย EH4404

## วิธีติดตั้งระบบ

### สิ่งที่ต้องเตรียม  
- **Termux** (แอป)

###  ติดตั้งแอป
:  
```bash  
pkg update && pkg upgrade -y
pkg install python git -y
```

###  ติดตั้งสคริปต์
:  
```bash  
git clone https://github.com/EH4404/smsfree
cd smsfree
pip install -r requirements.txt
python main.py
```
