# ยิงเบอร์ฟรี โดย EH4404

## วิธีติดตั้งระบบ

### สิ่งที่ต้องเตรียม  
- **Termux** (แอป)

###  ติดตั้งแอป
:  
```bash  
pkg update && pkg upgrade  
pkg install python git  
```

###  ติดตั้งสคริปต์
:  
```bash  
git clone https://github.com/username/EH4404-System.git  
cd EH4404-System
pip install -r requirements.txt
python eh4404.py
```
